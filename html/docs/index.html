Betaflight 飞控固件文档

本项目是一个基于 STM32F405 的飞行控制器固件，采用模块化设计，包含完整的传感器驱动、姿态解算、PID控制和通信协议支持。

快速开始： 选择左侧导航中的章节开始阅读详细的 API 文档。每个章节都包含完整的函数说明和使用示例。

文档章节

BSP层

板级支持包，包含 I2C、SPI、UART 通信接口和系统级功能。

传感器驱动

ICM42688P IMU、HMC5883L 磁力计、BMP280 气压计完整驱动。

控制算法

Mahony 姿态解算、PID 控制器、二阶 Butterworth 滤波器。

通信协议

ELRS/CRSF 接收机协议支持，支持 50-250Hz 更新率。

使用示例

完整的代码示例，包含传感器读取、姿态解算、PID控制等。

性能指标

传感器规格、控制性能、计算性能等详细指标。

主要特性

  • 硬件平台: STM32F405 (Cortex-M4, 168MHz)
  • 传感器支持:
    • ICM42688P (6轴IMU: 加速度计 + 陀螺仪)
    • HMC5883L (3轴磁力计)
    • BMP280 (气压计 + 温度计)
  • 姿态解算: Mahony 互补滤波算法
  • 控制算法: 基于 Betaflight 的三轴 PID 控制器
  • 通信协议: ELRS/CRSF 接收机协议
  • 滤波器: 二阶 Butterworth 滤波器 (LPF/Notch/BPF)
